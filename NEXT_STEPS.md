## 🚀 今晚需要用户操作的事项（按步骤）

### Step 1: 部署上线（最重要！）
**方案：Railway.app**（最省心，免费额度够用）

1. 访问 https://railway.app
2. 点击「Login with GitHub」→ 授权
3. 点击「New Project」→「Deploy from GitHub repo」
4. 选择 `qiyuan-studio/grassflow`
5. 在 Dashboard → Variables 添加：
   ```
   DEEPSEEK_API_KEY=sk-your-deepseek-api-key
   DEEPSEEK_BASE_URL=https://api.deepseek.com
   DEEPSEEK_MODEL=deepseek-v4-flash
   JWT_SECRET=grassflow-jwt-secret-2026
   JWT_EXPIRES_IN=7d
   NODE_ENV=production
   ```
6. 等待 2-3 分钟部署完成
7. 部署完成后会得到一个 URL（如 `grassflow.up.railway.app`）
8. **发给用户即可使用！**

> 备用方案：如果 Railway.app 注册麻烦，也可以用 **Zeabur** (https://zeabur.com) 同样支持 GitHub 一键部署，中文界面更友好。

### Step 2: 收款方式选择（二选一）

**方案A：LemonSqueezy（推荐）**
- 全球流行的 SaaS 收款平台
- 支持支付宝/微信/信用卡
- 需要注册账号，绑定银行账户
- 自动处理发票、退款、订阅管理
- 注册: https://lemonsqueezy.com

**方案B：支付宝当面付**
- 需要企业支付宝或个体工商户
- 接入简单，费率低 (0.6%)
- 需要提供营业执照（或使用个人收款码先跑着）

**方案C：先用微信人工收款（最快）**
- 我提供一个微信号，用户直接转账
- 在后台手动激活 License Key（已开发好）
- 后续再接入自动支付

### Step 3: 自定义域名（可选，但有域名更专业）

1. 购买域名（阿里云/腾讯云/Cloudflare 约 ¥30-80/年）
2. 在 Railway 项目 Settings → Domains 中绑定
3. 配置 DNS 解析
